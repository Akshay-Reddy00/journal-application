## Journal Application

----

- This repo contains the source code for Journal Application, a tool designed to helps users to express their thoughts and feelings more clearly.
- By providing a private space for journaling, the application aims to support users in gaining control over their emotions and improve mental well-being.

<br>

----

- Secure User Authentication
